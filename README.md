# Descripción
StreamingLab es una plataforma la cual le permite al usuario manejar sus cuentas de streaming de manera rapida y segura, proteger su cuenta de usuarios desconocidos y administracion total de sus cuentas.
# Funciones
Se puede Regsitrar y Ver Planes en la Pantalla de Inicio, se debe ingresar el Nombre,Correo y Contraseña.
Dentro de los Planes puede ver las distintas Ofertas. 
Si el Usuario desea Contratar un plan, se le redirigira a la pagina de Pago
Dentro de la pagina de pago, el usuario debera ingresar sus datos bancarios
